## 网站简介

本系统是一个恋爱心情发布系统，该系统包含有两种不同权限的用户：

* **普通用户**可以在系统上发布、浏览阅读恋文。
* **管理员**可以对系统上的数据进行管理。

## 技术实现

* **前端：**  HTML CSS JS, JQuery Bootstrap框架 
* **后端：**  tornado, python, 
* **数据库：** mysql
